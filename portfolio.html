主な機能

  • スムーズなプレイヤーコントロールシステム
  • 状態機械を使用した敵AIの実装
  • Rigidbodyベースの物理演算
  • カメラコントロールとターゲティングシステム
  • パーティクルエフェクトとアニメーション

使用技術

Unity 2022 C# 3D Graphics Physics Animation

PROJECT 03

Multiplayer System Implementation

概要

Photon Engineを使用したリアルタイムマルチプレイヤーシステムの実装。最大4人での同時プレイを実現しました。

主な機能

  • リアルタイムプレイヤー同期
  • ルームマッチングシステム
  • ラグ補償とネットワーク最適化
  • ボイスチャット統合
  • リコネクション機能

使用技術

Photon Engine Unity C# Networking Multiplayer

成果

平均レイテンシ50ms以下を実現。安定した同期システムにより、スムーズなマルチプレイヤー体験を提供しています。
